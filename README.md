# Car Sales Dashboard – Power BI Project
## Overview
This project is a clean and interactive Power BI dashboard built for a car dealership company. It helps analyze sales performance by showing key metrics, weekly and monthly trends, and category-wise insights. The dashboard allows the business to easily track revenue, number of cars sold, pricing patterns, and regional performance.

## Objective
The objective of this project is to design a user-friendly dashboard that provides clear insights into:
- Year-to-Date (YTD) performance
- Month-to-Date (MTD) performance
- Year-over-Year (YOY) comparison
- Sales patterns across body styles, colors, regions, and brands
This helps the company make informed and data-driven decisions.

## Dataset Information
The dataset includes detailed information for each car sold:
- Date of Sale
- Car Model
- Body Style
- Color
- Company/Brand
- Dealer Region
- Selling Price
- Sales Amount
- Number of Cars Sold
These fields are used to calculate KPIs and build visual insights.

## KPI Requirements
### Sales KPIs
- YTD Total Sales
- MTD Total Sales
- YOY Growth in Total Sales
- Difference between YTD Sales and PTYD Sales
### Average Price KPIs
- YTD Average Price
- MTD Average Price
- YOY Growth in Average Price
- Difference between YTD Avg Price and PTYD Avg Price
### Cars Sold KPIs
- YTD Cars Sold
- MTD Cars Sold
- YOY Growth in Cars Sold
- Difference between YTD Cars Sold and PTYD Cars Sold

## Charts and Visuals
- YTD Weekly Sales Trend (Line Chart)
- YTD Sales by Body Style (Pie Chart)
- YTD Sales by Color (Pie Chart)
- YTD Cars Sold by Dealer Region (Map Chart)
- Company-wise YTD Sales Table
- Detailed Sales Table showing all car sales records

## Dashboard Layout
- Top Section: KPIs for YTD, MTD, YOY, and differences
- Middle Section: Line chart, pie charts, and map visual
- Bottom Section: Company-wise sales table and full detailed sales table
This layout provides both high-level and detailed insights in a structured manner.

## Tools Used
- Power BI Desktop
- DAX for calculations
- Power Query for data cleaning

## Output Summary
The final dashboard clearly visualizes the company’s sales performance, product preferences, regional demand, and yearly comparison. It enables the dealership to monitor progress, identify trends, and make better business decisions using real-time insights.
